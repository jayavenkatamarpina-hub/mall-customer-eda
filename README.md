# Mall Customers EDA

Data cleaning and exploratory analysis on the Mall Customers dataset from Kaggle (200 customers, features: Gender, Age, Annual Income, Spending Score).

## What's in this notebook

- Checked for missing values, duplicates, and outliers
- Visualized distributions, correlations, and group comparisons across gender
- Engineered two new features (Age Group and Spending-to-Income Ratio) to help a future model
- Summarized the main patterns found in the data

- `Mall_Customers_EDA.ipynb` — the notebook
- `Mall_Customers.csv` — the dataset
